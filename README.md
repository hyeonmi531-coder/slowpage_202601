# [Slow Page]

(http://s1.slowpage.shop/)

> 책과 함께 여유로운 시간을 보내는 공간, 슬로우 페이지

## 프로젝트 개요

**Slow Page**는 책을 사랑하는 사람들을 위한 웹사이트로, 월간 구독 서비스와 큐레이션 도서 추천을 제공합니다.

### 주요 기능

- 메인 비주얼 슬라이드 (PC/모바일 반응형)
- 텍스트 marquee(캐러셀) + 영상 (키프레임)
- 인기 태그 검색 (텍스트 타이핑효과(키프레임) + 아코디언 UI)
- 뉴스 섹션 - (호버시 영역확장 + 슬라이드)
- 책 속 밑줄 - 형광펜효과 (스크롤 트리거 Hight Text ) + 슬라이드
- 베스트 추천 (GSAP 가로 스크롤 애니메이션 + 3D 북 플립 + 시네마틱 줌-인(영상) +스티키 스크롤)
- 슬로우페이지 랭킹 (GSAP 스크롤 다이나믹 stagger (타임라인 효과) 애니메이션)

## 웹 표준, 접근성, SEO

### 1. 웹 표준 (Web Standards)

- **시맨틱 마크업(Semantic Markup)**: header, nav, section, article, footer 등의 태그를 사용하여 브라우저가 콘텐츠의 의미를 정확히 파악하도록 문서의 구조를 논리적으로 설계했습니다.
- **반응형 레이아웃**: 모바일과 데스크탑 이미지를 구분하여 사용(m-main3.jpg 등)하고, 유연한 레이아웃을 통해 다양한 디바이스에 대응합니다.
- **최신 이미지 포맷 활용**: .webp 포맷을 사용하여 이미지 품질은 유지하면서 용량을 최적화해 웹 로딩 속도를 높였습니다.

### 2. 웹 접근성 (Web Accessibility)

- **대체 텍스트 제공**: alt="플라워 컬러링", alt="2026 트웰브저니"와 같이 이미지에 적절한 대체 텍스트를 부여하여 스크린 리더 사용자가 정보를 이해할 수 있도록 했습니다.
- **명확한 헤딩 구조**: h1~h3 등의 태그를 단계별로 사용하여 콘텐츠의 위계질서를 잡았습니다.
- **직관적인 아이콘 레이블**: 카트, 벨, 돋보기 등 아이콘 이미지에도 alt 속성을 넣어 기능을 설명했습니다.

### 3. SEO (검색 엔진 최적화)

- **시맨틱 마크업 및 구조화**: h1~h3 태그를 전략적으로 사용하여 '월간 구독', '슬로우 페이지 랭킹' 등 핵심 키워드의 위계질서를 확립했습니다.
- **메타 데이터 및 Open Graph 최적화**: `<title>`, `<meta name="description">`, OG 태그를 상세화하여 검색 노출 및 SNS 공유 시 클릭률을 높였습니다.
- **Canonical URL 설정**: 대표 URL을 명시하여 검색 엔진의 중복 콘텐츠 인식을 방지했습니다.
- **외부 채널 연동**: 푸터에 공식 SNS(인스타그램, 유튜브)를 연동하여 사이트 신뢰도를 강화했습니다.

## 주요 인터랙션

### 1. 헤더 스크롤 효과

- 스크롤 시 고정 헤더 배경색 변경

### 2. 마키(Marquee) 텍스트

- 섹션2 무한 흐르는 텍스트 애니메이션

### 3. 타이핑 효과

- 섹션4 인기 태그 텍스트 순차 타이핑 (@keyframes blinkCursor)

### 4. 아코디언 UI

- PC: hover 시 열림
- Mobile: 클릭 시 열림

### 5. GSAP 가로 스크롤

- 섹션7 책 열림 → 영상 확대 → 카드 슬라이드

### 6. 랭킹 GSAP 타임라인

- 섹션8 스크롤에 따른 순차 등장 애니메이션

### 7. 그 외

- @keyframes fadeLoop: 헤더 최상단 라인배너
- @keyframes videoBreath, wavePulse: 비디오의 확대축소 효과
- @keyframes floating: book 아이템의 위아래로 둥둥 흘러가는 효과
